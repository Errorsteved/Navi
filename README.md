# 物联网智能无人驾驶小车系统 / IoT-Based Autonomous Mini Vehicle System
## 概述 Overview

本项目构建了一套基于ROS 2的智能无人驾驶小车系统，实现多传感器感知、定位融合、局部路径规划与运动控制等核心功能。系统采用模块化架构，结合LiDAR、摄像头、GPS与IMU等传感器，适用于自主导航算法的研究与验证。

This project implements an IoT-oriented autonomous mini vehicle built on ROS 2, integrating multi-sensor perception, fused localization, local motion planning, and vehicle control. The system is designed with a modular structure and supports LiDAR, camera, GPS, and IMU sensors, serving as a compact platform for autonomous navigation research.
